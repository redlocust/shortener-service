Redlocust API Basejump: URL Shortener Microservice

User stories: I can pass a URL as a parameter and I will receive a shortened URL in the JSON response. When I visit that shortened URL, it will redirect me to my original link.

Example creation usage:
http://shortener-service-redlocust.c9users.io:8080/new/https://www.google.com
http://shortener-service-redlocust.c9users.io:8080/new/http://foo.com:80

Example creation output
{ "original_url":"http://foo.com:80",
"short_url":"http://shortener-service-redlocust.c9users.io:8080/8170" }

Usage:
http://shortener-service-redlocust.c9users.io:8080/2871
Will redirect to:
https://www.google.com/
